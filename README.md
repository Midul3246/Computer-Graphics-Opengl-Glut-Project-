# Computer-Graphics-Opengl-Glut-Project-
This project presents a virtual scenario of Comilla University developed using OpenGL and GLUT. The project features an interactive environment where a user-controlled car can be operated through keyboard inputs. Various animated elements such as moving clouds, an airplane, and trees with wind effects create a dynamic and realistic atmosphere. Additionally, mouse click interactions are implemented, including a music button that plays audio when clicked, enhancing the overall user experience and interactivity of the simulation.
